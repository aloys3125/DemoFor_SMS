Project Overview

This project showcases Smart Meeting Summarizer, a full-stack web application built using HTML5, CSS3, JavaScript for the frontend and Python (Flask) for the backend.
It leverages NLP with Hugging Face’s BART model to
convert meeting transcripts into clean, actionable summaries.
Due to cloud hosting limitations with large NLP models, this site includes a screen recording of the live application running locally, ensuring all features are visible.
The demo highlights my web development skills, including responsive design, UI/UX focus, and integration of complex backend logic—reflecting my ability to build and deploy complete, user-friendly web solutions.
